void Simulation::display ( )
virtual

Displays objects.

This function takes objects from obstacleArray and displays them in the OpenGL window by calling the drawing functions from drawing.cpp. The function then checks to make sure that the robot does not overlap the obstacles or target. The location of the robot is reset with random coordinates until it doesn't overlap any obstacles or the target. Then draws and displays robot.

Reimplemented from BaseGfxApp.

void Simulation::pause ( int  id)
static

Pauses simulation.

Pauses simulation by toggling flag. This prevents the environment from running the update function by returning at the beginning. This prevents objects from updating any attributes, effectively "pausing" the simulation.
